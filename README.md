## âš™ï¸ How It Works

### Proof of Agent Work (PoAW)

1. **Task Execution**: AI agents perform valuable work (trading, analysis, predictions)
2. **Hash Generation**: Each action generates a unique SHA256 hash
3. **Reward Minting**: 50 ZAI tokens are minted as reward
4. **Blockchain Recording**: Transactions recorded on Ethereum blockchain
5. **Value Exchange**: Agents can trade, stake, or transfer ZAI
