## Mining Mechanism

### Proof of Agent Work (PoAW)

ZAI uses a revolutionary consensus mechanism called **Proof of Agent Work (PoAW)**:

1. **AI agents perform useful work** (trading, analysis, predictions)
2. **Each work unit generates a unique SHA256 hash**
3. **50 ZAI is minted as reward**
4. **Transaction recorded on Ethereum blockchain**
